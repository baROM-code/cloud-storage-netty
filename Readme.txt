Cloud Storage by baROM

------------------------------------------------------
Облачное хранилище реализовано в виде клиент/сервер

Папка client
  Клиент (java nio)
  запуск client.java
 
Папка server
  Сервер (java netty)
  запуск server.java

Папка storage
 Каталоги и файлы пользователей
